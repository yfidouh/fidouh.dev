# fidouh.dev

Monorepo for everything deployed under `fidouh.dev`. Each app under `apps/` is a self-contained project deployed to its own subdomain via a dedicated Cloudflare Pages project.

| App          | Domain             | Stack                              |
| ------------ | ------------------ | ---------------------------------- |
| `apps/yahya` | `yahya.fidouh.dev` | Astro (static) + Tailwind v4 + Bun |
